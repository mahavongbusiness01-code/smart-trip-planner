# ✈️ Smart Trip Planner

**Smart Trip Planner** เป็นเว็บแอปพลิเคชันรูปแบบ Single Page Application (SPA) ที่ช่วยให้คุณวางแผนทริปท่องเที่ยว ค้นหาพิกัดและวางแผนเดินทางร่วมกับ AI และแชร์แผนให้กับเพื่อนๆ ได้ฟรี ผ่านฐานข้อมูลออนไลน์ Firebase Firestore หรือแชร์ลิงก์ด่วนผ่าน URL Hash

---

## 🌟 ฟีเจอร์หลัก (Key Features)

*   **Premium Glassmorphism UI/UX**: การออกแบบที่สวยงาม ทันสมัย รองรับทั้งคอมพิวเตอร์และอุปกรณ์เคลื่อนที่ (Mobile Responsive)
*   **Interactive Leaflet Map**: แสดงพิกัดและวาดเส้นทางการเดินทางในแต่ละวันโดยอัตโนมัติ พร้อมรองรับการคลิกบนแผนที่เพื่อระบุพิกัดโดยตรง
*   **Gemini AI Helper**: ช่วยค้นหา ละติจูด ลองจิจูด งบประมาณ แนะนำช่วงเวลาเดินทาง และคำแนะนำของสถานที่ต่างๆ ด้วย AI และเติมข้อมูลลงฟอร์มทันที
*   **Firebase Cloud Storage (ฟรี 100%)**: บันทึกข้อมูลทริปขึ้นฐานข้อมูลคลาวด์และรับลิงก์แชร์สั้นๆ (Trip ID) ส่งต่อให้เพื่อนเข้าดูและช่วยแก้ไขได้
*   **URL-Based Sharing (Base64 Fallback)**: แชร์ทริปแบบไม่ต้องมีฐานข้อมูล แปลงข้อมูลเข้ารหัสไว้ใน URL เพื่อให้ส่งต่อได้ทันที
*   **Budget Dashboard & Analytics**: แสดงกราฟวงกลมแยกแยะหมวดหมู่ค่าใช้จ่าย และแจ้งเตือนทันทีเมื่อยอดรวมเกินงบที่ตั้งไว้
*   **Local File Import/Export**: เซฟทริปลงเครื่องคอมพิวเตอร์ในรูปแบบไฟล์ `.json` และอัปโหลดไฟล์กลับขึ้นมาแก้ไขได้ตลอดเวลา

## 🛠️ วิธีการติดตั้งและใช้งานเบื้องต้น (Getting Started)

คุณสามารถเปิดใช้งานแอปพลิเคชันได้โดยตรงผ่านเว็บเบราว์เซอร์:
1. ดับเบิลคลิกเปิดไฟล์ `index.html` บนเบราว์เซอร์ของคุณ หรือรัน local server
2. แนะนำให้เปิดใช้งานผ่าน Local Web Server (เช่น Live Server ของ VS Code หรือใช้คำสั่ง Python/Node.js ด้านล่าง) เพื่อให้รองรับฟังก์ชันการทำลิงก์แชร์ที่ถูกต้อง

### วิธีรันด้วย Python (รันเพื่อจำลอง Web Server ท้องถิ่น):
เปิด Terminal ในโฟลเดอร์นี้แล้วพิมพ์:
```bash
python -m http.server 8000
```
จากนั้นเปิดบราวเซอร์ไปที่: `http://localhost:8000`

## 🔑 วิธีการตั้งค่า API Key และ ฐานข้อมูลออนไลน์ฟรี (Configuration)

หากต้องการใช้งาน **ระบบค้นหาด่วนด้วย AI (Gemini)** และ **ระบบบันทึกคลาวด์ (Firebase Firestore)** คุณสามารถเปิดแอปแล้วคลิกปุ่ม **"เมนูขวา (ปุ่มขีดสามขีด)" -> เลือก "ตั้งค่าฐานข้อมูล & AI"**

### 1. วิธีการรับ Gemini API Key ฟรี
1. ไปที่เว็บไซต์ [Google AI Studio](https://aistudio.google.com/)
2. เข้าสู่ระบบด้วยบัญชี Google ของคุณ
3. คลิกปุ่ม **"Get API key"** จากนั้นสร้าง API Key ใหม่
4. คัดลอก API Key มากรอกในช่อง "Gemini API" ในหน้าตั้งค่าของแอป

### 2. วิธีการสมัครและเปิดใช้งาน Firebase Firestore ฟรี (ฐานข้อมูลฟรี)
1. ไปที่เว็บไซต์ [Firebase Console](https://console.firebase.google.com/)
2. คลิก **"Add Project"** เพื่อสร้างโปรเจกต์ใหม่ (ตั้งชื่อโปรเจกต์ตามต้องการ เช่น `my-trip-planner` และกดดำเนินการฟรี)
3. ในหน้า Dashboard ของโปรเจกต์ คลิกสัญลักษณ์ **Web `</>`** เพื่อเริ่มสร้างเว็บแอปพลิเคชัน
4. ตั้งชื่อเว็บแอปพลิเคชันของคุณ และกดลงทะเบียน จากนั้น Firebase จะแสดงโค้ด `firebaseConfig` ให้คัดลอกเฉพาะข้อมูลในวงเล็บปีกกา เช่น:
   ```json
   {
     "apiKey": "AIzaSy...",
     "authDomain": "...",
     "projectId": "...",
     "storageBucket": "...",
     "messagingSenderId": "...",
     "appId": "..."
   }
   ```
5. นำ Config ไปวางในช่อง "ฐานข้อมูลออนไลน์ Firebase (Firestore)" ในหน้าตั้งค่าของแอป
6. **การเปิดใช้งาน Firestore Database**:
   - ในเมนูด้านซ้ายของ Firebase Console ไปที่ **Build -> Firestore Database**
   - คลิกปุ่ม **"Create Database"**
   - เลือกเซิร์ฟเวอร์ที่อยู่ใกล้เคียง (เช่น `asia-southeast1` สิงคโปร์ หรือ `asia-east1`)
   - เลือกเริ่มต้นในโหมดทดสอบ (Start in test mode) หรือตั้งค่า Security Rules ให้ทุกคนสามารถอ่านและบันทึกทริปได้ดังนี้:
     ```javascript
     rules_version = '2';
     service cloud.firestore {
       match /databases/{database}/documents {
         match /trips/{document} {
           allow read, write: if true;
         }
       }
     }
     ```
   - กด **Publish** เพื่ออัปเดตรูทความปลอดภัย

เมื่อกด **บันทึกการตั้งค่า** ในแอปเรียบร้อยแล้ว ปุ่ม **"เซฟขึ้นคลาวด์"** จะพร้อมทำงานทันที! ข้อมูลทั้งหมดจะถูกส่งเข้าฐานข้อมูลของท่านอย่างปลอดภัย และท่านสามารถแชร์ลิงก์ทริปให้เพื่อนดูและแก้ไขได้ฟรีตลอดไป
